## 🔍 Реалізовані алгоритми виявлення

### DoS атака
Виявляється при перевищенні порогової кількості пакетів від одного джерела у межах визначеного часового вікна.

**Критерій:** кількість пакетів > `DOS_THRESHOLD` за `TIME_WINDOW` секунд

### Port Scan
Виявляється, якщо одне джерело звертається до великої кількості різних портів.

**Критерій:** кількість унікальних портів > `PORTSCAN_THRESHOLD`

### Brute Force
Виявляється при великій кількості спроб підключення до сервісного порту (наприклад, SSH – порт 22).

**Критерій:** кількість звернень до порту 22 > `BRUTEFORCE_THRESHOLD`

## ⚙️ Налаштування

Усі параметри задаються в файлі `config.py`:

```python
# Файл з трафіком
PCAP_FILE = "/tshark/01.pcap"

# Порогові значення
DOS_THRESHOLD = 1000
PORTSCAN_THRESHOLD = 20
BRUTEFORCE_THRESHOLD = 15
TIME_WINDOW = 10

# Email-налаштування
SMTP_SERVER = "mail.example.com"
SMTP_PORT = 587
SMTP_USER = "admin@example.com"
SMTP_PASSWORD = "password"
ADMIN_EMAIL = "admin@example.com"
