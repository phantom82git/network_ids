# Network Intrusion Detection System (IDS)

## Опис проекту

Даний проєкт реалізує систему виявлення мережевих атак (Intrusion Detection System), що виконує аналіз мережевого трафіку з файлу формату `.pcap` та визначає потенційні атаки на основі статистичних та сигнатурних правил.

Система розроблена в межах дипломної роботи та демонструє модульну архітектуру аналізатора мережевого трафіку.

---

## Мета системи

Автоматизований аналіз мережевого трафіку з метою виявлення:

- DoS (Denial of Service) атак
- Сканування портів (Port Scan)
- Brute Force атак

---

## Принцип роботи

Система працює у наступній послідовності:

1. Зчитування мережевого трафіку з `.pcap` файлу
2. Аналіз кожного TCP/IP пакета
3. Формування статистики по IP-адресах
4. Виявлення аномалій
5. Логування результатів
6. Відправка email-сповіщення адміністратору

## Реалізовані алгоритми виявлення

### DoS атака

Виявляється при перевищенні порогової кількості пакетів від одного джерела у межах визначеного часовго вікна.

Критерій:

кількість пакетів > DOS_THRESHOLD за TIME_WINDOW секунд


---

### Port Scan

Виявляється, якщо одне джерело звертається до великої кількості різних портів.

Критерій:

кількість унікальних портів > PORTSCAN_THRESHOLD


---

### Brute Force

Виявляється при великій кількості спроб підключення до сервісного порту (наприклад, SSH – порт 22).

Критерій:

кількість звернень до порту 22 > BRUTEFORCE_THRESHOLD


---

## Налаштування

Усі параметри задаються в файлі `config.py`:

```python
PCAP_FILE = "/tshark/01.pcap"

DOS_THRESHOLD = 1000
PORTSCAN_THRESHOLD = 20
BRUTEFORCE_THRESHOLD = 15

TIME_WINDOW = 10

Запуск проекту
1. Встановити залежності
pip install scapy
2. Вказати шлях до pcap-файлу у config.py
PCAP_FILE = "/path/to/your/01.pcap"
3. Запустити систему
python3 main.py


## Логування

Результати роботи зберігаються у файл:

ids_log.txt

Формат запису:

[2026-03-05 12:30:01] DoS attack detected from 192.168.1.10
